# Tech Stack - Allan's Dotfiles

## Core Orchestration
- **Dotfile Management:** [chezmoi](https://www.chezmoi.io/)
- **Package Management:** [Homebrew](https://brew.sh/) (via `Brewfile`)

## Environment
- **Shell:** Fish Shell
- **Terminal Multiplexer:** Tmux
- **Editor:** Neovim (Lua-based)
- **Window Management:** Aerospace (Tiling Window Manager for macOS)

## Scripting & Automation
- **System Scripts:** POSIX Shell / Bash (e.g., `bootstrap.sh`, `macos-defaults.sh`)
- **Shell Customization:** Starship (Prompt)

## Languages & Development Tools
- **Python:** Managed with `uv` and linted with `ruff`
- **Go:** General development
- **Utilities:** ripgrep, fzf, jq, gitui
